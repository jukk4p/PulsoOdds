# 🏆 Diccionario Maestro de Ligas y Competiciones - PulsoOdds

Este diccionario sirve como fuente de verdad para la normalización de los nombres de las ligas y competiciones que provienen de la API externa. El objetivo es mostrar nombres en español profesional y estético en la web.
