## Project Standards (auto-resolved)

### Next.js & React
- Use App Router patterns.
- Prefer Server Components for data fetching.
- Use Tailwind CSS v4 utility classes.
- Follow the "breaking changes" guide for Next.js 16/15 as per `AGENTS.md`.

### Responsive Design
- **MANDATORY**: Mobile-first approach. Base styles for mobile, media queries for larger screens (`sm`, `md`, `lg`, etc.).
- Use high-density layouts (`PickRow`) with strict vertical alignment.

### UI/UX
- Theme: "Electric Neon" (vibrant colors, glassmorphism).
- Icons: `lucide-react`.
- Data Parsing: Use `parseJsonList` for stringified lists in DB.
